# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

You can report security issues by:

- Opening a GitHub Issue  
- Or contacting the project maintainer directly  

Please include as much detail as possible:

- Description of the vulnerability  
- Steps to reproduce  
- Affected versions  
- Potential impact  

Do **not** publicly disclose security issues before they are reviewed and addressed.

## Supported Versions

Only the latest version of this project is actively supported with security updates.

## Response Process

Security reports will be reviewed as soon as possible.  
We aim to respond in a reasonable timeframe and keep reporters informed about the status.

Thank you for helping to keep this project secure.
